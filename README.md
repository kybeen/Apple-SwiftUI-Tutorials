### [ CoreMotionTest ]
- Core Motion 프레임워크를 사용해서 기기의 센서값을 받아오는 것을 공부한 내용입니다.
- Watch Connectivity 프레임워크를 함께 사용해서 watchOS 앱에서 수집한 센서값들을 iOS 앱에 불러와서 확인해보았습니다.
